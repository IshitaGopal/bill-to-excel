# 
An application to convert bank statements into excel. Includes function to extract bank statements from:
1. Revolut 
2. Amex 

## Project Structure 

```
pdf-to-excel/
├── extractors/
│   ├── amex.py          # Amex extraction logic
│   └── revolut.py       # Revolut extraction logic
├── main.py              # Run the app
├── requirements.txt     # Dependencies
└── output/              # Where Excel files go (optional folder)
```
